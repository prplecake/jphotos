# jphotos

[![builds.sr.ht
status](https://builds.sr.ht/~mjorgensen/jphotos.svg)](https://builds.sr.ht/~mjorgensen/jphotos?)

jphotos is a simple HTTP server for photo albums.

## Features

* No "social media" features
* No JavaScript!
* Blazingly fast
